COMMITEN
--------
Um die Version reibungslos zu aktualisieren gibt es ein Paar Dinge die zu beachten sind:

Schritt 1: commit
  Dabei bitte immer spezifizieren was genau ihr dabei seid hochzuladen.
  Anschließend NICHT "pushen"

Schritt 2: PULL
  So holt ihr die aktuelle Version aus dem ROOT-Repository. GIT wird erstmal sicherstellen das es keine
  Konflite gibt.
  Erst wenn alles grün ist solltet ihr pushen.
